# 导出文本功能修复文档

## 问题描述

点击"导出文本"按钮时，提示"请先选择一个页面"，无法正常导出内容。

## 问题根源

### 当前逻辑

```javascript
const currentPage = store.currentDiary.pages.find(p => p.id === editingPageId.value)
if (!currentPage) {
  showCopyToast('请先选择一个页面')
  return
}
```

### `editingPageId` 的设置时机

| 场景 | `editingPageId` 值 | 结果 |
|------|-------------------|------|
| 浏览模式翻页查看 | `null` | 导出失败 |
| 双击进入编辑模式 | 当前页面 ID | 导出成功 |

### 问题原因

用户在**浏览模式**下只是翻页查看内容，没有双击进入编辑模式，`editingPageId` 始终为 `null`，导致导出失败。

## 修复方案

### 方案 C：混合方案

- 编辑模式下：导出正在编辑的页面
- 浏览模式下：导出当前查看的页面

## 修改内容

### 修改前

```javascript
async function handleExport() {
  if (!store.currentDiary) return

  const currentPage = store.currentDiary.pages.find(p => p.id === editingPageId.value)
  if (!currentPage) {
    showCopyToast('请先选择一个页面')
    return
  }

  const title = store.currentDiary.title
  const content = pageContents.value[String(currentPage.id)] || currentPage.content || '（空白页）'
  const date = formatDateTime(currentPage.updated_at) || '未记录'
  const quote = getRandomQuote()

  const exportText = `《${title}》\n${content}\n${date}\n${quote}`

  try {
    await navigator.clipboard.writeText(exportText)
    showCopyToast('已复制内容到剪贴板')
  } catch (err) {
    console.error('复制失败:', err)
    const textArea = document.createElement('textarea')
    textArea.value = exportText
    document.body.appendChild(textArea)
    textArea.select()
    document.execCommand('copy')
    document.body.removeChild(textArea)
    showCopyToast('已复制内容到剪贴板')
  }
}
```

### 修改后

```javascript
async function handleExport() {
  if (!store.currentDiary) return

  let currentPage = null

  // 优先使用编辑中的页面
  if (editingPageId.value) {
    currentPage = store.currentDiary.pages.find(p => p.id === editingPageId.value)
  }

  // 如果没有编辑中的页面，使用当前查看的页面
  if (!currentPage) {
    const pageIndex = currentPageIndex.value
    // 排除封面(0)和封底(最后)
    if (pageIndex > 0 && pageIndex <= contentPageCount.value) {
      currentPage = store.currentDiary.pages[pageIndex - 1]
    }
  }

  if (!currentPage) {
    showCopyToast('当前页面无内容可导出')
    return
  }

  const title = store.currentDiary.title
  const content = pageContents.value[String(currentPage.id)] || currentPage.content || '（空白页）'
  const date = formatDateTime(currentPage.updated_at) || '未记录'
  const quote = getRandomQuote()

  const exportText = `《${title}》\n${content}\n${date}\n${quote}`

  try {
    await navigator.clipboard.writeText(exportText)
    showCopyToast('已复制内容到剪贴板')
  } catch (err) {
    console.error('复制失败:', err)
    const textArea = document.createElement('textarea')
    textArea.value = exportText
    document.body.appendChild(textArea)
    textArea.select()
    document.execCommand('copy')
    document.body.removeChild(textArea)
    showCopyToast('已复制内容到剪贴板')
  }
}
```

## 核心修改说明

### 页面索引对应关系

| `currentPageIndex` | 页面类型 | `contentPageCount` 对应 |
|-------------------|----------|------------------------|
| 0 | 封面 | 不导出 |
| 1 | 内容页 1 | pages[0] |
| 2 | 内容页 2 | pages[1] |
| ... | ... | ... |
| N | 内容页 N | pages[N-1] |
| N+1 | 封底 | 不导出 |

### 判断逻辑

```javascript
// pageIndex > 0：排除封面
// pageIndex <= contentPageCount.value：排除封底
if (pageIndex > 0 && pageIndex <= contentPageCount.value) {
  currentPage = store.currentDiary.pages[pageIndex - 1]
}
```

## 行为对比

| 场景 | 修改前 | 修改后 |
|------|--------|--------|
| 浏览模式查看内容页 | 提示"请先选择一个页面" | 导出当前页面内容 |
| 浏览模式查看封面 | 提示"请先选择一个页面" | 提示"当前页面无内容可导出" |
| 浏览模式查看封底 | 提示"请先选择一个页面" | 提示"当前页面无内容可导出" |
| 编辑模式编辑内容页 | 导出正在编辑的页面 | 导出正在编辑的页面 |
