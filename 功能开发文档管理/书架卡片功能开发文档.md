# 书架卡片功能开发文档

## 功能概述

在 `BookshelfView.vue` 页面中实现两个交互式卡片：

1. **写作统计 + 今日心情卡片**（浅色卡片）
2. **名言互动卡片**（深色卡片，点击变色显示名言）

---

### 数据需求

| 数据项 | 来源 | 说明 |
|--------|------|------|
| 本周写作页数 | 后端统计 | 7天内更新的页面数 |
| 连续打卡天数 | 后端统计 | 连续有写作记录的天数 |
| 今日心情 | 前端 localStorage | 用户选择的心情 |

### 后端接口设计

#### 新增接口：`GET /api/v1/diaries/stats`

**文件位置**：`/server/routes/diaries.js`

```javascript
router.get('/stats', authMiddleware, (req, res) => {
  const db = req.db;
  const userId = req.user.id;
  
  // 本周写作页数（7天内更新的页面）
  const weeklyPages = db.prepare(`
    SELECT COUNT(*) as count 
    FROM pages p 
    JOIN diaries d ON p.diary_id = d.id 
    WHERE d.user_id = ? 
    AND p.updated_at >= datetime('now', '-7 days', 'localtime')
  `).get(userId).count;
  
  // 连续打卡天数（简化版：统计最近有写作的天数）
  const streakDays = db.prepare(`
    SELECT COUNT(DISTINCT date(updated_at)) as days
    FROM pages p
    JOIN diaries d ON p.diary_id = d.id
    WHERE d.user_id = ?
    AND p.updated_at >= datetime('now', '-30 days', 'localtime')
    ORDER BY date(updated_at) DESC
  `).get(userId).days;
  
  res.json({
    code: 200,
    msg: '获取成功',
    data: {
      weekly_pages: weeklyPages,
      streak_days: streakDays
    }
  });
});
```

**文件位置**：`/client/src/views/BookshelfView.vue`

```vue
<script setup>
// 添加以下代码

const stats = ref({
  weekly_pages: 0,
  streak_days: 0
})

const moods = [
  { id: 'happy', icon: 'sentiment_very_satisfied', label: '开心', color: 'bg-amber-100' },
  { id: 'calm', icon: 'sentiment_satisfied', label: '平静', color: 'bg-green-100' },
  { id: 'neutral', icon: 'sentiment_neutral', label: '一般', color: 'bg-gray-100' },
  { id: 'sad', icon: 'sentiment_dissatisfied', label: '低落', color: 'bg-blue-100' },
  { id: 'down', icon: 'sentiment_very_dissatisfied', label: '难过', color: 'bg-indigo-100' }
]

const todayMood = ref(null)

async function loadStats() {
  try {
    const data = await getStats()
    stats.value = data
  } catch (error) {
    console.error('加载统计失败:', error)
  }
}

function loadTodayMood() {
  const today = new Date().toDateString()
  const saved = localStorage.getItem('todayMood')
  if (saved) {
    const { date, mood } = JSON.parse(saved)
    if (date === today) {
      todayMood.value = mood
    }
  }
}

function selectMood(moodId) {
  todayMood.value = moodId
  localStorage.setItem('todayMood', JSON.stringify({
    date: new Date().toDateString(),
    mood: moodId
  }))
}

// 在 onMounted 中调用
onMounted(async () => {
  // ... 现有代码
  await loadStats()
  loadTodayMood()
})
</script>

<template>
<!-- 替换现有的浅色卡片 -->
<div class="bg-surface-container-lowest p-6 rounded-xl custom-shadow">
  <div class="flex items-center justify-between mb-4">
    <h4 class="font-headline text-lg text-primary flex items-center gap-2">
        <span class="material-symbols-outlined">edit_note</span>
        本周写作
      </h4>
    <span class="text-xs text-on-surface-variant/60">{{ new Date().toLocaleDateString('zh-CN', { month: 'long', day: 'numeric' }) }}</span>
  </div>
  
  <div class="space-y-3 mb-6">
    <div class="flex items-center justify-between">
      <span class="text-sm text-on-surface-variant">已写</span>
      <span class="font-headline text-2xl text-primary">{{ stats.weekly_pages }} <span class="text-sm font-body">页</span></span>
    </div>
    <div class="flex items-center justify-between">
      <span class="text-sm text-on-surface-variant">连续打卡</span>
      <span class="font-headline text-lg text-primary flex items-center gap-1">
          {{ stats.streak_days }} 天
          <span class="material-symbols-outlined text-sm text-amber-500">local_fire_department</span>
        </span>
    </div>
  </div>
  
  <div class="border-t border-outline-variant/20 pt-4">
    <h5 class="font-label text-xs uppercase tracking-widest text-on-surface-variant mb-3 flex items-center gap-1">
        <span class="material-symbols-outlined text-sm">mood</span>
        今日心情
      </h5>
    <div class="flex justify-between">
      <button
        v-for="mood in moods"
        :key="mood.id"
        @click="selectMood(mood.id)"
        :class="[
          'w-10 h-10 rounded-full flex items-center justify-center transition-all duration-300',
          todayMood === mood.id 
            ? `${mood.color} ring-2 ring-primary ring-offset-2 scale-110` 
            : 'bg-surface-container hover:bg-surface-container-high'
        ]"
      >
        <span class="material-symbols-outlined text-xl">{{ mood.icon }}</span>
      </button>
    </div>
    <p v-if="todayMood" class="text-xs text-center text-on-surface-variant mt-3">
      今日心情：{{ moods.find(m => m.id === todayMood)?.label }}
    </p>
  </div>
</div>
</template>
```

### 名言库设计

**存储位置**：前端（名言库固定，约25条）

**理由**：
- 名言库不大，前端存储响应更快
- 不需要频繁更新
- 后续可扩展为后端存储（收藏、统计功能）

### 颜色方案

点击变色使用柔和的深色系过渡：

| 变体 | 背景色 | 文字色 |
|------|--------|--------|
| 默认 | `bg-inverse-surface` | `text-inverse-on-surface` |
| 变体1 | `bg-slate-800` | `text-slate-100` |
| 变体2 | `bg-zinc-800` | `text-zinc-100` |
| 变体3 | `bg-stone-800` | `text-stone-100` |
| 变体4 | `bg-neutral-800` | `text-neutral-100` |
