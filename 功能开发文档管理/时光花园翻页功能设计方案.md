# 一、功能概述

## 1.1 核心概念

将存档页面改造为"时光花园"（或"回望"），包含两个视图：
- **第一页：时光存档**（原有存档功能）
- **第二页：心灵花园**（AI Reflection 功能）

用户通过翻页动画，从"时光存档"进入"心灵花园"，营造仪式感和私密感。

---

## 1.2 设计理念

### **仪式感**
- 翻页本身就是一种仪式，用户从"存档"翻页进入"AI Reflection"
- 强化"进入私人空间"的感觉

### **私密感**
- "心灵花园"强化私密属性
- AI 内容隐藏在翻页后的第二页，不会过度突出

### **AI 可选**
- 用户需要主动翻页才能看到 AI 内容
- 符合"AI Optional"理念

# 三、翻页效果方案

## 3.1 翻页方式：自动翻页 + 微细提示

### **实现方案**
- 用户点击"探索花园"按钮后，自动触发翻页动画
- 翻页动画持续 1-2 秒，营造仪式感
- 翻页完成后，自动进入心灵花园页面

### **微细提示**
- 在按钮附近添加提示文字："翻开日记，探索内心"
- 提示文字默认半透明（opacity: 0.4），hover 时变为完全可见
- 提示文字使用 Material Icons 的 `auto_stories` 图标，增强视觉引导

### **优点**
- 体验流畅，减少用户操作
- 可以通过动画控制翻页速度，营造仪式感
- 微细提示引导用户点击，不会让用户困惑

---

## 3.2 翻页动画技术：Three.js + StPageFlip

### **技术选型**
- **Three.js：** 用于创建 3D 翻页场景，渲染翻页动画
- **StPageFlip：** 项目已有的翻页库，用于实现真实的笔记本翻页效果

### **实现思路**
1. **第一层：Three.js 3D 场景**
   - 创建一个 3D 书本模型，用户点击按钮后，书本开始翻页
   - 翻页动画使用 Three.js 的 3D 变换（rotateY）

2. **第二层：StPageFlip 真实翻页**
   - 在 Three.js 场景中叠加 StPageFlip 的翻页效果
   - StPageFlip 提供真实的纸张弯曲和阴影效果

3. **第三层：Vue 页面过渡**
   - 翻页动画完成后，使用 Vue Router 跳转到心灵花园页面
   - 跳转时添加过渡动画（fade 或 slide），确保视觉连贯

### **技术优势**
- **Three.js：** 提供 3D 视觉效果，增强仪式感
- **StPageFlip：** 提供真实的纸张效果，增强沉浸感
- **Vue Router：** 页面结构清晰，易于维护

# 四、页面结构方案

## 4.1 双页面结构（路由跳转 + 翻页动画过渡）

### **路由设计**
```javascript
// router.js
const routes = [
  {
    path: '/reflection',  // 导航栏名称：回望
    name: 'Reflection',
    component: () => import('@/views/ReflectionView.vue'), // 主容器
    children: [
      {
        path: '',
        name: 'ArchiveSection',
        component: () => import('@/views/ArchiveSection.vue'), // 第一页：时光存档
        meta: { page: 1 }
      },
      {
        path: 'garden',
        name: 'GardenSection',
        component: () => import('@/views/GardenSection.vue'), // 第二页：心灵花园
        meta: { page: 2 }
      }
    ]
  }
]
```

### **页面结构**
```
ReflectionView.vue (主容器)
├── ArchiveSection.vue (第一页：时光存档)
│   ├── 书架视图（原存档内容）
│   ├── 探索按钮（右下角）
│   ├── 微细提示（"翻开日记，探索内心"）
│   └── Three.js + StPageFlip 翻页动画层
│
└── GardenSection.vue (第二页：心灵花园)
    ├── AI Reflection 内容
    ├── 返回按钮（左下角）
    └── 花园背景装饰
```

## 4.2 翻页动画流程

### **流程图**
```
用户点击"探索花园"按钮

```

# 六、技术实现细节

## 6.1 Three.js 3D 翻页场景

### **场景初始化**
```javascript
import * as THREE from 'three'

// 创建场景
const scene = new THREE.Scene()
const camera = new THREE.PerspectiveCamera(45, window.innerWidth / window.innerHeight, 1, 10000)
const renderer = new THREE.WebGLRenderer({ alpha: true })

// 创建书本模型
const bookGeometry = new THREE.BoxGeometry(window.innerWidth, window.innerHeight, 10)
const bookMaterial = new THREE.MeshPhongMaterial({ color: 0xffffff, transparent: true })
const book = new THREE.Mesh(bookGeometry, bookMaterial)

scene.add(book)
camera.position.set(0, 0, 1000)
```

### **翻页动画**
```javascript
function flipToGarden() {
  // 动画参数
  const duration = 2000 // 2 秒
  const targetRotation = Math.PI // 180 度

  // 使用 GSAP 控制动画
  gsap.to(book.rotation, {
    y: targetRotation,
    duration: duration,
    ease: 'power2.inOut',
    onUpdate: () => {
      renderer.render(scene, camera)
    },
    onComplete: () => {
      // 动画完成，触发路由跳转
      router.push('/reflection/garden')
    }
  })
}
```

## 6.2 StPageFlip 真实翻页效果

### **初始化**
```javascript
import { PageFlip } from 'st-page-flip'

const pageFlip = new PageFlip(document.getElementById('book-container'), {
  width: window.innerWidth,
  height: window.innerHeight,
  maxShadowOpacity: 0.5,
  showPageCorners: true,
  useMouseEvents: false, // 禁用鼠标事件，使用按钮触发
  flippingTime: 2000 // 翻页时间 2 秒
})

pageFlip.loadFromImages([
  'archive-page-image', // 第一页（存档）
  'garden-page-image'   // 第二页（心灵花园）
])
```

### **触发翻页**
```javascript
function flipToGarden() {
  pageFlip.flipNext() // 自动翻到下一页
}
```

# 八、风险评估

## 8.1 技术风险

| 风险项 | 影响 | 解决方案 |
|--------|------|---------|
| Three.js 性能问题 | 移动端可能出现卡顿 | 使用 CSS 3D 变换作为降级方案 |
| StPageFlip 集成复杂度 | 需要额外学习成本 | 先使用 CSS 动画，后续集成 StPageFlip |
| 路由跳转动画不连贯 | 可能影响用户体验 | 使用过渡动画协调页面跳转 |

---

## 8.2 产品风险

| 雕塑项 | 影响 | 解决方案 |
|--------|------|---------|
| 用户不理解翻页概念 | 可能错过 AI 功能 | 微细提示 + 探索按钮图标引导 |
| 翻页动画过长 | 可能让用户等待不耐烦 | 控制动画时长在 1-2 秒内 |
| 导航栏名字理解偏差 | 用户可能不知道"回望"的含义 | 添加副标题或图标提示 |
