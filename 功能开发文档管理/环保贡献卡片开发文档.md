# 环保贡献卡片功能开发文档

## 功能概述

在 `BookshelfView.vue` 页面的 `bg-surface-container-lowest p-6 rounded-xl custom-shadow` 卡片中添加**环保贡献**模块，展示用户的写作数据对应的环保价值。

## 计算公式

### 环保换算标准

```
1 页纸 ≈ 4 克
1 kg 纸张 ≈ 2.4 kg CO₂ 排放
1 棵树 ≈ 可生产 8500 页纸
```

### 前端计算逻辑

```javascript
// 计算环保数据
function calculateEcoStats(totalPages) {
  // 节省纸张（克）
  const paperGrams = totalPages * 4
  // 节省纸张（千克，保留2位小数）
  const paperKg = (paperGrams / 1000).toFixed(2)
  // 减少CO2排放（千克，保留2位小数）
  const co2Kg = (paperGrams / 1000 * 2.4).toFixed(2)
  // 保护树木数量（保留2位小数）
  const trees = (totalPages / 8500).toFixed(2)
  
  return {
    paperKg,
    co2Kg,
    trees
  }
}
```

**修改后**：

```javascript
router.get('/stats', authMiddleware, (req, res) => {
  const db = req.db;
  const userId = req.user.id;

  // 本周写作页数
  const weeklyPages = db.prepare(`
    SELECT COUNT(*) as count 
    FROM pages p 
    JOIN diaries d ON p.diary_id = d.id 
    WHERE d.user_id = ? 
    AND p.updated_at >= datetime('now', '-7 days', 'localtime')
  `).get(userId).count;

  // 连续打卡天数
  const streakDays = db.prepare(`
    SELECT COUNT(DISTINCT date(updated_at)) as days
    FROM pages p
    JOIN diaries d ON p.diary_id = d.id
    WHERE d.user_id = ?
    AND p.updated_at >= datetime('now', '-30 days', 'localtime')
    ORDER BY date(updated_at) DESC
  `).get(userId).days;

  // 累计总页数
  const totalPages = db.prepare(`
    SELECT COUNT(*) as count
    FROM pages p
    JOIN diaries d ON p.diary_id = d.id
    WHERE d.user_id = ?
  `).get(userId).count;

  res.json({
    code: 200,
    msg: '获取成功',
    data: {
      weekly_pages: weeklyPages,
      streak_days: streakDays,
      total_pages: totalPages
    }
  });
});
```

## 环保数据来源说明

| 数据 | 来源 | 说明 |
|------|------|------|
| 1页纸重量 | 行业标准 | A4纸约4-5克，取4克作为估算值 |
| CO₂排放系数 | 环保机构数据 | 生产1kg纸张约产生2.4kg CO₂ |
| 树木产纸量 | 林业数据 | 1棵成年树可生产约8500页A4纸 |

> 注：以上数据为估算值，仅供参考，实际数据可能因纸张类型、生产工艺等因素有所不同。
