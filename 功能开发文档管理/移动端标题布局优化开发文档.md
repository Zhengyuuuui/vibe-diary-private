# 移动端标题布局优化开发文档

## 需求概述

在移动端（< 768px）将日记标题从 header 移动到 main 区域顶部，并添加装饰效果。桌面端保持现有布局不变。

## 布局对比

### 桌面端（≥ 768px）- 保持不变
┌─────────────────────────────────────────────────────────────┐
│
 │
│   ┌─────────┐   ┌─────────────┐   ┌─────────────────────┐   │
│   │ ← 书架  │   │ 日记标题    │   │ [浏览|编辑] + 新页  │   │
│   └─────────┘   └─────────────┘   └─────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
│ │
│   ┌─────────────────────────────────────────────────────┐   │
│   │              page-flip-container                    │   │
│   └─────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘

### 移动端（< 768px）- 新布局
┌─────────────────────────────────────┐
│
 │
│   ┌─────────┐   ┌───────────────┐   │
│   │ ← 书架  │   │ [浏览|编辑]   │   │
│   └─────────┘   └───────────────┘   │
└─────────────────────────────────────┘
│ │
│   ┌─────────────────────────────┐   │
│   │  ═══════  日记标题  ═══════ │   │  ← 新增标题区域（带装饰）
│   └─────────────────────────────┘   │
│   ┌─────────────────────────────┐   │
│   │    page-flip-container      │   │
│   └─────────────────────────────┘   │
└─────────────────────────────────────┘

## 修改方案

### 1. 修改 Header 模板

**位置**：`<header>` 内部

```vue
<header class="bg-transparent backdrop-blur-xl sticky top-0 z-50">
  <div class="flex justify-between items-center w-full px-6 py-4 max-w-4xl mx-auto">
    <!-- 返回按钮 -->
    <button @click="goBack" class="flex items-center space-x-2 text-on-surface-variant hover:text-on-surface transition-colors">
      <span class="material-symbols-outlined">arrow_back</span>
      <span class="font-label text-xs hidden md:inline">书架</span>
    </button>
    
    <!-- 标题：桌面端显示，移动端隐藏 -->
    <div class="font-headline text-lg text-on-surface tracking-tight hidden md:block">
      {{ store.currentDiary?.title || '加载中...' }}
    </div>
    
    <!-- 操作按钮 -->
    <div class="flex items-center space-x-3">
      <div class="flex items-center bg-surface-container rounded-lg p-1">
        <button
          @click="toggleToBrowseMode"
          :class="[
            'flex items-center space-x-1 px-3 py-1.5 rounded-md font-label text-xs transition-all',
            !isEditMode
              ? 'bg-primary text-on-primary shadow-sm'
              : 'text-on-surface-variant hover:text-on-surface'
          ]"
        >
          <span class="material-symbols-outlined text-sm">visibility</span>
          <span>浏览</span>
        </button>
        <button
          @click="toggleToEditMode"
          :class="[
            'flex items-center space-x-1 px-3 py-1.5 rounded-md font-label text-xs transition-all',
            isEditMode
              ? 'bg-primary text-on-primary shadow-sm'
              : 'text-on-surface-variant hover:text-on-surface'
          ]"
        >
          <span class="material-symbols-outlined text-sm">edit</span>
          <span>编辑</span>
        </button>
      </div>
      
      <!-- 新页按钮：桌面端显示，移动端隐藏 -->
      <button 
        @click="addNewPage" 
        class="hidden md:flex items-center space-x-1 text-on-surface-variant hover:text-on-surface transition-colors"
      >
        <span class="material-symbols-outlined">note_add</span>
        <span class="font-label text-xs">新页</span>
      </button>
      
      <!-- 新页按钮：移动端只显示图标 -->
      <button 
        @click="addNewPage" 
        class="md:hidden p-2 text-on-surface-variant hover:text-on-surface transition-colors"
      >
        <span class="material-symbols-outlined">note_add</span>
      </button>
    </div>
  </div>
</header>
```

### 2. 修改 Main 模板 - 添加移动端标题区域

**位置**：`<main>` 内部，`page-flip-container` 之前

```vue
<main class="flex-1 flex flex-col items-center justify-center py-8 px-4">
  <!-- 加载状态 -->
  <div v-if="!store.currentDiary" class="text-center">
    <span class="material-symbols-outlined animate-spin text-4xl text-primary">refresh</span>
    <p class="mt-4 font-label text-sm text-on-surface-variant">正在翻开日记本...</p>
  </div>

  <template v-else>
    <!-- 移动端标题区域 -->
    <div class="md:hidden w-full max-w-md mb-4 px-4">
      <div class="flex items-center justify-center">
        <div class="flex-1 h-px bg-on-surface-variant/20"></div>
        <h1 class="font-headline text-base text-on-surface tracking-wide px-4">
          {{ store.currentDiary.title }}
        </h1>
        <div class="flex-1 h-px bg-on-surface-variant/20"></div>
      </div>
    </div>

    <!-- 翻页容器 -->
    <div class="page-flip-container w-full flex justify-center">
      <!-- ... 原有内容 ... -->
    </div>
  </template>
</main>
```

## 完整修改清单

### 修改 1：Header 标题添加响应式隐藏

```vue
<!-- 修改前 -->
<div class="font-headline text-lg text-on-surface tracking-tight">
  {{ store.currentDiary?.title || '加载中...' }}
</div>

<!-- 修改后 -->
<div class="font-headline text-lg text-on-surface tracking-tight hidden md:block">
  {{ store.currentDiary?.title || '加载中...' }}
</div>
```

### 修改 2：Header 返回按钮文字响应式隐藏

```vue
<!-- 修改前 -->
<button @click="goBack" class="flex items-center space-x-2 ...">
  <span class="material-symbols-outlined">arrow_back</span>
  <span class="font-label text-xs">书架</span>
</button>

<!-- 修改后 -->
<button @click="goBack" class="flex items-center space-x-2 ...">
  <span class="material-symbols-outlined">arrow_back</span>
  <span class="font-label text-xs hidden md:inline">书架</span>
</button>
```

### 修改 3：新页按钮响应式处理

```vue
<!-- 修改前 -->

  <span class="material-symbols-outlined">note_add</span>
  <span class="font-label text-xs">新页</span>
</button>
<!-- 移动端：只显示图标 -->

  @click="addNewPage" 
  class="md:hidden p-2 text-on-surface-variant hover:text-on-surface transition-colors"

  <span class="material-symbols-outlined">note_add</span>
</button>
```

### 修改 4：Main 区域添加移动端标题

```vue
<!-- 修改前 -->
<main class="flex-1 flex items-center justify-center py-8 px-4">
  <div v-if="!store.currentDiary" class="text-center">
    ...
  </div>
  <div v-else class="page-flip-container w-full flex justify-center">
    ...
  </div>

<!-- 修改后 -->
<main class="flex-1 flex flex-col items-center justify-center py-8 px-4">
  <div v-if="!store.currentDiary" class="text-center">
    ...
  </div>
  
  <template v-else>
    <!-- 移动端标题区域 -->
    <div class="md:hidden w-full max-w-md mb-4 px-4">
      <div class="flex items-center justify-center">
        <div class="flex-1 h-px bg-on-surface-variant/20"></div>
        <h1 class="font-headline text-base text-on-surface tracking-wide px-4">
          {{ store.currentDiary.title }}
        </h1>
        <div class="flex-1 h-px bg-on-surface-variant/20"></div>
      </div>
    </div>
    
    <div class="page-flip-container w-full flex justify-center">
      ...
    </div>
  </template>
</main>
```

## 响应式断点说明

| Tailwind 类 | 含义 | 屏幕宽度 |
|-------------|------|----------|
| `md:hidden` | 在 ≥768px 时隐藏 | 移动端显示 |
| `hidden md:block` | 默认隐藏，≥768px 显示 | 桌面端显示 |
| `hidden md:flex` | 默认隐藏，≥768px flex | 桌面端显示 |
| `hidden md:inline` | 默认隐藏，≥768px inline | 桌面端显示 |

## 注意事项

1. **main 布局调整**：将 `flex items-center justify-center` 改为 `flex flex-col items-center justify-center`，以支持垂直排列

2. **标题区域间距**：使用 `mb-4` 与翻页容器保持适当间距

3. **最大宽度限制**：标题区域使用 `max-w-md` 限制最大宽度，避免过宽

4. **保持功能一致**：所有按钮功能保持不变，只是布局调整
